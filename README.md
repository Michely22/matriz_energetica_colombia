## 📖 Descripción del Proyecto

Este proyecto forma parte del **Curso de Talento Tech - Análisis de Datos**, donde se desarrolla un dashboard interactivo para visualizar y analizar la matriz energética de Colombia. 

El sistema permite monitorear indicadores clave como generación, demanda, costos, inversiones e impacto ambiental (emisiones de CO2) de diferentes fuentes de energía a través del tiempo.

**🎓 Contexto Académico:**
- **Curso:** Talento Tech - Análisis de Datos
- **Módulo:** Bases de Datos + Visualización
- **Herramientas:** SQL, Python, Streamlit, Plotly

## 📊 Características del Dashboard

### 🏠 Landing Page
- Presentación profesional del proyecto
- Información sobre objetivos y tecnologías

### 📈 Panel de Control Principal
- **KPIs en tiempo real** (Generación, Demanda, Cobertura, Inversión)
- **Gráficos interactivos** con Plotly
- **Filtros dinámicos** por tipo de energía y año
